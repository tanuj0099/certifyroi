# PRD

## Product
Name: CertifyROI  
One-line summary: A premium India-focused certification ROI platform that helps users evaluate whether a certification is worth the money, time, and effort.

## Core experience
- Landing page establishes the mountain / route / summit metaphor.
- Tools page lists all available tools.
- Each tool has its own dedicated page.
- Theme toggle should work consistently across all pages.
- Design should feel premium, editorial, clean, and cohesive.

## Goals
- Help users evaluate certification ROI clearly.
- Maintain strong visual consistency across all pages.
- Keep the experience mobile-friendly and readable in both light and dark modes.

## Core pages
- Landing page
- Tools page
- Individual tool pages
- FAQ page
- Any supporting information pages

## Requirements
- Restore removed tools on tools page.
- Each tool must have an individual page/route.
- Footer must be present where intended.
- FAQ content must render properly.
- No stray broken markup or visual artifacts.
- Buttons and layout system should match the landing-page design language.
- Internal pages should visually harmonize with landing page.
- Keep content intact unless it is clearly broken, nonsensical, or placeholder text.

## Non-functional requirements
- Responsive layout
- Strong text contrast
- Theme consistency
- Clean typography and spacing
- No AI-slop visual patterns
